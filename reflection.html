Reflection

The process of designing and building my website was difficult, due to a small timeframe and numerous other assessments during similar time periods. A key challenge was redoing the HTML as the sample website was built fast and lots of it was forgotten. However, I overcame this challenge by researching and googling how to do tasks such as creating a box around elements, using css to put space between, and decorating the website better with CSS. Another challenge was doing the CSS as I briefly learnt it in class and forgot lots of it. I overcame this by researching and learning what each element such as box-shadow, display, and margin worked, and how I could use them to make the website cooler. I also had to adapt to situations such as making websites look cooler and fixing profile pictures multiple times. I fixed this issue by using ai to help fix issues and troubleshoot as well as learning and experimenting with different designs and colours to get one I liked.

I’ve learned that HTML provides the structure and core of the website, CSS decorates the website, VS Code is a platform where you can code and create different websites and games, GitHub as a place where you host websites and collaborate for repositories, and Git is a distributed version control system.

I’ve learned that computing isn’t all about memorising how to do each line of code, it’s about learning and constantly googling for information.

For the future, I want to improve my website by adding javascript code and improving the ux for the website.
